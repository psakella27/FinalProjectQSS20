# FinalProjectQSS20
Repository for Pranav Akella's QSS20 final project.
The main questions I hope to answer are questions surrounding antisocial behavior,
hatred, and political rhetoric on Twitter or potentially other platforms during the pandemic.
Some potential questions are: did hateful rhetoric against specific groups (gender, racial,
religious) increase during the pandemic? Did mention of health issues increase? Did certain
fringe communities (incel sphere, looksmaxxers, etc.) have more activity?
